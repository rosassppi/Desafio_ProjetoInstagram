# Desafio - Projeto Instagram

Projeto desenvolvido como desafio do curso Fullstack PRO do Sujeito Programador.

## Tecnologias
- HTML
- CSS

## Objetivo
Praticar a estilização e estruturação de uma página com login e cadastro utilizando o Instagram como modelo.

## Como usar
Basta abrir o arquivo `index.html` no navegador.
